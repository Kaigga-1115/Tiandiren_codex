# 後端啟動說明

這個資料夾是教學用的 Flask 後端，負責保存寶可夢夥伴資料與傳說紀錄資料。

## 一、先安裝需要的套件

請先確認電腦有安裝 Python 3。

如果你已經有 Python，可以進入 `backend/` 後安裝套件：

```bash
conda activate Tiandiren
cd backend
pip install -r requirements.txt
```

如果你的電腦還沒有 Python，請先安裝 Python 3，再回來執行上面的指令。

## 二、如何啟動後端

在 `backend/` 資料夾中執行：

```bash
conda activate Tiandiren
python app.py
```

如果你的系統預設是 `python3`，也可以使用：

```bash
python3 app.py
```

## 三、如何確認後端有成功啟動

啟動成功後，終端機會顯示類似這樣的訊息：

```text
Running on http://127.0.0.1:5050
```

你也可以打開瀏覽器，直接測試：

```text
http://127.0.0.1:5050/health
```

如果畫面出現：

```json
{"ok": true, "message": "backend is running"}
```

就表示後端有成功啟動。

## 四、後端提供的 API

### 寶可夢夥伴

- `GET /api/companions` 取得所有寶可夢夥伴
- `GET /api/companions/<id>` 取得單一寶可夢夥伴詳細資訊
- `POST /api/companions` 新增寶可夢夥伴

### 傳說紀錄

- `GET /api/records` 取得所有傳說紀錄
- `GET /api/records/<id>` 取得單一傳說紀錄詳細資訊
- `POST /api/records` 新增傳說紀錄

## 五、資料保存在哪裡

目前資料會存在：

- `backend/data.json`

這樣設計的好處是：

- 初學者比較容易理解
- 不需要先學資料庫
- 新增後會真的存到檔案裡

## 六、如果前端要讀取後端資料

前端可以用 `fetch()` 向後端 API 取資料。

例如：

```js
fetch("http://127.0.0.1:5050/api/companions")
```

之後如果要把前端畫面改成直接讀後端資料，就可以用這個方式連接。
